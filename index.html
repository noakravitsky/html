MY AUTOMATION PROJECT:


This project was developed to demonstrate automation testing scenarios based on the Shicor.il website.

Shicor.il is a tourism application combined of databases with artificial intelligence and advanced algorithms to provide the best experience for booking trips.

The project is built according to the Page Object Model (POM) method- a design pattern, commonly used in test automation that creates Object Repository for web UI elements.

The advantage of the model is that it reduces code duplication and improves test maintenance.
